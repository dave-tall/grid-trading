종목을 추가하세요
종목명, 진입가, 수량을 입력하면
그리드 매매 스케줄이 자동 생성됩니다

📌 매매 규칙

  • 총 투자금 ₩1,000,000 한도
  • 첫 진입: 지정 가격에 원하는 수량 매수
  • 추가 매수: 첫 진입가 기준 1% 하락마다 ₩10,000어치
  • 매도: 각 매수가 기준 2% 상승 시 해당 수량만 매도

🔄 사용 방법

  • 종목명, 진입가, 수량 입력 후 "스케줄 생성"
  • 현재가 입력 → 업데이트 → 🔔 강조 행 확인
  • 매수·매도 체결 시 해당 버튼으로 완료 처리
  • 최대 10개 종목 동시 관리 가능

🎨 색상 안내

  • ● 녹색 — 현재가 기준 조건 충족 행
  • ● 회색 — 완료 처리된 거래
  • ● 파란색 — 대기 중 주문

📲 앱 설치 방법

  • 안드로이드 Chrome: 우상단 메뉴(⋮) → "홈 화면에 추가"
  • 삼성 인터넷: 메뉴 → "페이지 추가" → "홈 화면"
  • iOS Safari: 하단 공유(□↑) → "홈 화면에 추가"
